# Emotion Detection System

This project is a web-based application developed as part of the IBM "Introduction to AI Engineering with Google Gemini" course. 
It uses Watson NLP to detect emotions (anger, disgust, fear, joy, and sadness) in text provided by the user.

## Features
- Real-time emotion analysis via IBM Watson API.
- Web interface built with Flask.
- Static code analysis with PyLint (10/10 rating).
- Unit tests for reliable performance.
